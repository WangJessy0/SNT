8-Le Web et ses Fonctionnements

1. Que désigne ‘Le Web’ ?

Le web désigne un ensemble de ressources liées les unes aux autres par des liens hypertextes, c'est-à-dire tous les éléments sur lesquels on clique pour passer d'une page à une autre.

2. Quelle est la condition essentielle pour votre ordinateur, pour que vous puissiez accéder au web ?

Pour pouvoir y accéder, une machine doit être connectée à internet.

3. Où sont stockées toutes les ressources du web ?

Toutes les ressources du web sont stockées sur des serveurs distants.

4. Expliquer le processus des échanges client-serveur.

Pour pouvoir y accéder et les consulter, un client doit envoyer des requêtes à un serveur, qui peut lui répondre en retour en lui fournissant des codes de pages web, des images, des vidéos, etc. Ces échanges entre les clients et les serveurs sont régis par des protocoles.

5. Nommer des protocoles qui régissent les échanges entre le client et le serveur.

Le protocole HTTP notamment, ou sa version sécurisée, le protocole HTTPS.

6. Qu’est-ce qu’un URL ? De quoi est-il composé (Détailler via un exemple) ?

Les ressources présentes sur le web sont localisées par un URL, il s'agit d'une chaîne de caractères qui indique le protocole utilisé, le nom de domaine, et le chemin à suivre dans une arborescence pour accéder à la ressource sur le serveur.

7. Qu’est-ce qu’un moteur de recherche ?

Les moteurs de recherche sont des outils spécifiques du web destinés à fournir aux internautes des URL en adéquation avec les recherches qu'ils souhaitent effectuer.

8. Expliquer l’usage et le fonctionnement des robots Crawler ou Spider ?

Ces moteurs de recherche utilisent pour cela des robots appelés crawlers ou spiders. Ils parcourent les ressources du web de liens hypertextes en lien hypertexte, afin de les analyser et d’en indexer le contenu.

9. Expliquer comment un moteur de recherche propose des URL à partir de Mots clés de l’internaute.

Un internaute qui souhaite accéder à une ressource web dont il ne connaît pas l’URL utilise un moteur de recherche en lui soumettant une requête sous la forme d'une série de mots clés. En recoupant ces mots clés avec sa base de données et à l'aide d'algorithmes, le moteur de recherche renvoie à l'internaute une liste d'URL compatibles, triées selon leur pertinence. Les moteurs de recherche ne sont pas tous équivalents et diffèrent selon les algorithmes qu'ils utilisent, le traitement des requêtes des internautes et des contenus publicitaires mis en avant. En effet, pour être mise en avant, certaines entreprises choisissent un référencement payant.

10. Qu’est-ce qu’une page web, Que peut-elle contenir ? Nommer un langage pour créer une page web.

Une page web est une ressource particulière du web pouvant accueillir du texte, des images, des vidéos, des sons et de nombreuses animations. Ces contenus sont le plus souvent mis en forme de façon à rendre la navigation sur la page la plus aisée et la plus esthétique possible. Ces pages web sont codées à l'aide de différents langages.

11. Comment appelle-t-on le système utilisé par le langage HTML ?

Le langage HTML permet de créer et structurer le contenu de la page : texte, lien, images, titres, etc. Il s'agit d'un langage qui utilise un système de balises pour inclure tous ces éléments.

12. Qu’est-ce que le CSS ?

Le langage CSS permet de mettre en forme ces contenus. Il est utilisé dans des feuilles de style spécifiques, c'est-à-dire des fichiers envoyés par les serveurs en complément du code HTML, pour définir les polices utilisées, les couleurs, les tailles, les décorations, et une partie des animations visuelles. D'autres langages sont également utilisés afin de rendre les pages web dynamiques et interactives. De cette façon, il est possible de personnaliser une page web. Cette personnalisation peut se servir notamment des cookies, qui sont de petits fichiers de texte stockés par le navigateur de l'internaute et qui, en général, sont destinés à faciliter la navigation en retenant par exemple des identifiants et des mots de passe.

13. Est-ce que la réutilisation d’une page Web est toujours autorisée ? Nommer un modèle de licence.

Le contenu des pages web est soumis aux législations des états. Si leur publication ne nécessite pas d'autorisation ou de contrôle particulier au préalable, les contenus peuvent être signalés s'ils enfreignent la loi. D'autre part, la propriété intellectuelle s'applique également et la réutilisation de certains contenus n'est pas toujours autorisée. Les conditions d'utilisation sont indiquées par les auteurs, elles peuvent par exemple suivre le modèle des licences Creative Commons.
