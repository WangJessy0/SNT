2-Données structurées

Stockage, organisation et traitement des données

Comment stocker, organiser et traiter des données ?

Comment le numérique a-t-il fait évoluer leur importance dans nos pratiques ?

Une donnée est une valeur ou un ensemble de valeurs pouvant prendre plusieurs formes : des nombres ou des chaînes de caractères, et permettant de décrire un ou plusieurs objets. Une donnée représente une information.

Les données structurées désignent un ensemble de données stockées sous la forme d'une table, regroupant de multiples objets selon un ou plusieurs descripteurs. Le format le plus courant est le format CSV : il s'agit d'un fichier texte contenant des données séparées par un caractère spécial, appelé séparateur.

Ces fichiers peuvent être importés dans des tableurs. Par exemple, une table peut contenir quinze données relatives à trois objets différents : des élèves classés selon cinq descripteurs. Les données structurées peuvent être analysées à l'aide d'un ou plusieurs traitements.

À partir d’un tableur ou d’un langage de programmation, il est possible de réaliser de multiples opérations comme :

  • des tris,
  • des recherches,
  • des calculs statistiques,
  • des représentations graphiques.

Lorsque le volume de données est trop important, un tableur peut devenir lent. Il est alors préférable d’utiliser un langage de programmation comme Python, ou un langage de gestion de bases de données comme SQL.

Une métadonnée est une donnée relative à une autre donnée. Pour un fichier, cela peut inclure :

  • le nom,
  • le type,
  • la date de création,
  • le nom de l’auteur, etc.

Dans les photographies numériques, les métadonnées au format EXIF regroupent des informations telles que la date, le type d'appareil ou encore le mode de codage des couleurs.

Les données personnelles sont des informations confidentielles liées aux individus : nom, âge, genre, localisation, centres d’intérêt, état de santé, etc. Elles sont collectées massivement et souvent utilisées pour la publicité ciblée.

Les modes de stockage des données :

  • Stockage personnel :
    • Interne : disque dur, disque SSD
    • Externe : disque dur externe, carte SD
  • Stockage distant (cloud) :
    • Serveurs distants dans des datacenters
    • Accessibles depuis n’importe quelle machine connectée à Internet

Le stockage dans le cloud est très utilisé pour sa praticité, mais a un impact environnemental : consommation d’eau, d’électricité, et utilisation de métaux rares.

À l’échelle mondiale, la consommation d’électricité augmente de 3 % chaque année à cause du développement de ces infrastructures.
